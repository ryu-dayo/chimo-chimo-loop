HTML5動画に、PiP（ピクチャインピクチャ）、A-Bリピート、再生速度の変更、メディア統計情報などの機能を備えた、洗練された軽量なフローティング・コントロールバーを追加するユーザースクリプトです。

> 主に **macOS版 Safari** で動作確認を行っています。
> Chromium系ブラウザ（Edgeなど）にも対応していますが、サイトによっては機能や表示が異なる場合があります。

## 主な機能 (Features)

- 🌠 **ピクチャインピクチャ (PiP)**: ワンクリックで動画をフローティング再生。
- 🔁 **拡張ループ**: 全体ループとA-B区間リピートの両方に対応。
- ⚡️ **再生速度調整**: 動画の再生速度を自在に変更できます。
- 📸 **無劣化スクリーンショット**: UIオーバーレイなしで、現在のフレームを1:1の高品質なPNG画像としてワンクリックで保存。
- 📊 **メディア統計情報**: FPS、解像度、カラースペースをリアルタイムでトラッキング。
- 👻 **自動非表示 (オートハイド)**: 操作がない状態が3秒続くと、邪魔にならないよう自動的にフェードアウト。

## 使い方 (User Guide)

| アイコン | アクション | 説明 |
| :---: | :--- | :--- |
| 🔳 | **PiP** | ピクチャインピクチャモードのオン/オフを切り替えます。 |
| 🔁 | **ループ / A点設定** | 全体ループを有効にし、現在の再生位置を**A点（開始点）として自動的に設定**します。 |
| 🅱️ | **B点設定** | （ループ開始後に表示）クリックして**A-B区間**を確定し、区間リピートを開始します。 |
| 📸 | **スクリーンショット** | 現在のフレームを高品質なPNG画像として保存します。 |
| ⏩ | **詳細 / 再生速度** | メニューを展開して、再生速度の変更やメディア統計情報を確認します。 |

> **💡 ヒント:** **A-Bリピート**と**0.5倍速再生**を組み合わせれば、難しいギターソロの耳コピや、ダンスの振り付けの練習などに最適です。

## 互換性について (Compatibility)

- **ユニバーサルサポート**: 標準のHTML5 `<video>` 要素を使用しているほとんどのページで動作します。
- **ネイティブ機能の優先**: ウェブサイト自体が同様の機能を標準で提供している場合は、最高のユーザー体験を得るために、サイト内蔵のコントロールを使用することをお勧めします。
- **動作に関する注意事項**: 一部のプラットフォームでは独自の再生ロジックを採用しているため、スクリプトの表示状態と実際のサイトの動作に差異が生じる場合があります。
