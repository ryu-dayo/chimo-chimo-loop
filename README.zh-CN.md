# chimo-chimo-loop

一款轻量级的用户脚本，为 HTML5 视频新增悬浮控制条，提供画中画（PiP）、循环播放（Loop）与倍速控制功能。

> 主要在 **macOS 的 Safari** 上测试  
> 也可在 Chromium 系浏览器（如 Edge）正常运行，但部分网站的功能与表现可能存在差异。

## 功能特点

- 🌠 **画中画**：一键开启/关闭悬浮窗播放
- 🔁 **高级循环**：支持全片循环及 A-B 循环
- ⚡️ **倍速控制**：提供常用的倍速播放选项
- 🧊 **极简 UI**：自动注入，悬浮于视频左上角
- 👻 **自动隐藏**：无操作 3 秒后自动淡出

## 操作指南

| 图标 | 功能 | 说明 |
| :---: | :--- | :--- |
| 🔳 | **画中画** | 切换画中画模式 (PiP)。 |
| 🔁 | **开启循环 / 定 A 点** | 开启全片循环，并**自动将当前时间设为 A 点**。 |
| 🅱️ | **定 B 点** | (开启循环后出现) 点击后锁定 **A-B 区间**并开始循环。 |
| ❎ | **退出循环** | 清除 A/B 循环，并退出全片循环模式。 |
| ⏩ | **更多/倍速** | 展开倍速控制菜单与视频统计信息。 |

> **💡 小技巧：** 想要设置 A-B 循环时，只需点击一次“循环按钮”即可自动定下起点，等视频播放到区间终点时点击“🅱️”图标即可。

## 兼容性说明

- **广泛适用**：适用于大多数使用原生 HTML5 `<video>` 元素的页面。
- **功能优先级**：若目标网站已内置相同功能，建议优先使用网站原生功能以获得最佳兼容性。
- **行为说明**：某些网站采用非标准播放逻辑，可能导致脚本状态与实际行为不完全一致。
