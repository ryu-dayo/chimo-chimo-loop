一款轻量级的用户脚本，为 HTML5 视频新增功能强大的悬浮控制条，支持画中画、A-B 循环、倍速调节、无损截图及媒体信息统计。

> 主要在 **macOS 的 Safari** 上测试  
> 也可在 Chromium 系浏览器（如 Edge）正常运行，但部分网站的功能与表现可能存在差异。

## 功能特点

- 🌠 **画中画**：一键开启/关闭悬浮窗播放
- 🔁 **高级循环**：支持全片循环及 **A-B 区间重复**
- ⚡️ **倍速控制**：提供常用的倍速播放选项
- 📸 **无损截图**：一键截取 1:1 高清 PNG 画面，无 UI 遮挡
- 📊 **媒体统计**：实时显示帧率 (FPS)、分辨率、色彩空间等硬核数据
- 👻 **自动隐藏**：无操作 3 秒后自动淡出

## 操作指南

| 图标 | 功能 | 说明 |
| :---: | :--- | :--- |
| 🔳 | **画中画** | 切换画中画模式 (PiP)。 |
| 🔁 | **开启循环 / 定 A 点** | 开启全片循环，并**自动将当前时间设为 A 点**。 |
| 🅱️ | **定 B 点** | (开启循环后出现) 点击后锁定 **A-B 区间**。 |
| 📸 | **无损截图** | 将当前画面保存为高质量 PNG 图片。 |
| ⏩ | **更多/倍速** | 展开倍速控制菜单与视频统计信息。 |

> **💡 小技巧：** 配合 **A-B 循环**与 **0.5 倍速**，可以更轻松地通过视频扒谱或练习舞蹈动作。

## 兼容性说明

- **广泛适用**：适用于大多数使用原生 HTML5 `<video>` 元素的页面。
- **功能优先级**：若目标网站已内置相同功能，建议优先使用网站原生功能以获得最佳兼容性。
- **行为说明**：某些网站采用非标准播放逻辑，可能导致脚本状态与实际行为不完全一致。
